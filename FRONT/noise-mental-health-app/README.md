# Noise and Mental Health App

This project analyzes the relationship between environmental noise and mental health outcomes. It provides a user-friendly interface for users to explore data, visualize results, and make predictions based on noise levels and other environmental factors.

## Features

- **Landing Page**: A modern and interactive landing page that introduces the project and its objectives.
- **Data Analysis**: Tools for analyzing the impact of noise pollution on mental health.
- **Visualization**: Interactive visualizations to help users understand the data better.
- **Prediction Models**: Machine learning models to predict mental health outcomes based on noise data.

## Installation

1. Clone the repository:
   ```
   git clone https://github.com/yourusername/noise-mental-health-app.git
   ```
2. Navigate to the project directory:
   ```
   cd noise-mental-health-app
   ```
3. Install the required packages:
   ```
   pip install -r requirements.txt
   ```

## Usage

To run the app, execute the following command:
```
streamlit run app.py
```

## Directory Structure

- `src/`: Contains the source code for the app, including components, utilities, and models.
- `assets/`: Contains styles and configuration files.
- `data/`: Contains raw and processed data.
- `tests/`: Contains unit tests for the project.
- `requirements.txt`: Lists the required Python packages.
